hexo 博客引擎笔记

先清理再部署

$ hexo clean
$ hexo d

部署千万条,清理第一条.
流程不规范,找错两行泪.

使用图片

Markdown显示代码

格式

\```
代码
\```

注意是符号是 `,叫重音符或抑音符，又称反单引号，是西文字符中的附加符号，主要用于计算机相关领域.(百度百科)
不是单引号(别问我怎么知道的)

不要处理我的文章

如果你不想你的文章被处理，可以将layout: 设为 false 。

加密页面

使用 hexo-blog-encrypt

npm install hexo-blog-encrypt

在 Hexo 根目录的 _config.yml 添加:

encrypt:
    enable: true

在Markdown中添加

password: 你的密码

OK!

貌似不太支持中文密码 -> 文章加密测试

使用命令新建文章

普通文章
$ hexo new [layout:可选] 标题名 --path 文件名
草稿
$ hexo new draft 标题名 --path 文件名
发布(草稿转正)
$ hexo publish 文件名
